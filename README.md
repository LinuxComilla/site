Boilerplate for ES6 projects.

## Install

Install with [npm](https://www.npmjs.com/):

```sh
$ npm install --save boilerplate-es6
```

## What is es6-boilerplate

boilerplate-es6 is a template for your es6 based projects using

* gulp
* mocha & chai
* babel
